# Trainingplanner


## What?

A simple app for creating workout plans and share them



## Get started

```bash
npm i
npm run dev
```


## Technology

 - React
 - [React Router](https://reactrouter.com/en/main)
 - [Tailwind](https://tailwindcss.com/docs/installation)
 - [Daisy UI](https://daisyui.com/components)
